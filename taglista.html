Tag Lista

A tag <"ul"> (unordered list) é usada para criar uma lista não ordenada em HTML.
Uma lista não ordenada é uma lista de itens onde a ordem dos itens não é importante.
Cada item da lista é representado por um elemento <"li"> (list item).
Essa tag é frequentemente usada para criar menus, listas de recursos, etapas de um processo e muito mais.
