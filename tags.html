Tags <"h1"> a <"h6">, <"br">, <"p">, <"hr">

As tags <"h1"> a <"h6'> são usadas em HTML para criar cabeçalhos ou títulos com diferentes níveis de importância.
O <"h1"> é o cabeçalho de maior importância e representa o título principal da página, enquanto o <"h6"> é o cabeçalho de menor importância.
